# Realtime Subprocess

Show the output from subprocess in realtime.

Example:

```bash
python rt_proc.py -c "echo Started to run demo;sleep 0.5;echo 1;sleep 0.5;echo 2;sleep 0.5;echo 3;sleep 0.5;echo Finished"
```
