# Rich Exception

Show local variables and extra lines around where exception occurred.

## How to use

1. Install rich

```
pip install -U rich
```

2. Place `sitecustomize.py` in `site-packages` directory which can be found by running:

```
python -c "import sys;print(sys.path)"
```
